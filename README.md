## Requirements

REMOD is tested to work under Python 3.6, packages required include

- transformers==2.8.0
- torch==1.7.0+cu110
- numpy==1.18.3
- tqdm==4.45.0
- scipy==1.6.2
- joblib==0.13.2
- dgl_cu110==0.6.1
- dgl==0.6.1
- scikit_learn==0.24.2

All packages can be installed by `pip install -r requirements.txt`
